## eslint 通用配置（react）

### 安装

```bash
  npm install eslint --save-dev

  #airbnb eslint extends configs
  npm install eslint-config-airbnb --save-dev

  # 关闭与 prettier 冲突的规则
  npm install eslint-config-prettier --save-dev

```
