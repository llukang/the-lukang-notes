#### Auto Close Tag

  `html` 自动闭合标记

#### Auto Rename Tag

#### CSS Peek

## 工具插件

#### GitLens

跟踪代码显示的代码作者，提交搜索，历史记录和GitLens资源管理器

#### Git History

显示提交历史的精美图表

#### Git Project Manager

Git项目管理器

#### ESLint

`eslint` 代码检查工具

#### Live Server 

有静态和动态页面的实时重新加载功能的本地开发服务器
